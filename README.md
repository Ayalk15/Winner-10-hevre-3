# Winner 10 Hevre Premium v4

גרסת Premium v4 לאתר הניחושים.

## מה חדש
- עיצוב בית משודרג יותר.
- לוח בקרה מהיר למחזור.
- 10 משתתפים גנריים ללא שמות אישיים.
- טבלת דירוג חדשה עם מדליות.
- כרטיסי הישגים וסטטיסטיקות.
- בלי package-lock.json כדי ש־Vercel לא ייתקע.

## הרצה מקומית
```bash
npm install
npm run dev
```

## פריסה ב־Vercel
Build Command: npm run build
Output Directory: dist
