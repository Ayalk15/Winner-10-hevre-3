# Winner 10 Hevre – גרסת Premium

אפליקציית ניחושים לליגת העל: משחקים, ניחושים, ג׳וקר, סטטיסטיקות, טבלה, שעון ותאריך חיים ופאנל מנהל.

## חשוב
אין כרגע Firebase / Database / LocalStorage. הנתונים נשמרים רק בזיכרון של הדפדפן ולכן לאחר רענון הדף הכול מתאפס.

## הרצה מקומית
```bash
npm install
npm run dev
```

## בנייה לפרסום
```bash
npm run build
```
