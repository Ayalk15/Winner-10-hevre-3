# Winner 10 Hevre — Premium v6

מערכת ניחושים משודרגת: משתתפים, ניחושים, ג׳וקר, טבלה, סטטיסטיקות, אזור אישי ופאנל מנהל.

סיסמת מנהל: 2531

## הרצה
npm install
npm run dev

## בנייה
npm run build

## חשוב
בגרסה זו הנתונים נשמרים בדפדפן בלבד באמצעות localStorage. בשלב הבא ניתן לחבר Firebase כדי שכל המשתמשים יראו נתונים משותפים בזמן אמת.
