# Winner 10 Hevre — Premium v2

גרסה מתוקנת ומעוצבת לפרסום ב-Vercel.

## חשוב
אין Firebase ואין Database. הנתונים נשמרים רק בזמן שהדף פתוח, וברענון הכול מתאפס.

## תיקון מרכזי
Tailwind מוגדר לקרוא גם קבצים שנמצאים בשורש הפרויקט, לכן העיצוב נטען תקין ב-Vercel.

## פרסום
- Build Command: npm run build
- Output Directory: dist
